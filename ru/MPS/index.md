# Media Processing Service (MPS)


_Last updated: 2025-12-03 12:49_


Media Processing Service (MPS) — это облачный сервис для обработки мультимедийного контента, включая транскодирование аудио и видео, запись живых трансляций, наложение водяных знаков, захват экрана и AI-обработку медиа.

MPS предназначен для разработчиков и компаний, которым требуется масштабируемая, надёжная и гибкая обработка мультимедийных данных.

## Обзор

### Архитектура продукта

Вы можете загружать исходные видеофайлы в бакет COS через консоль, SDK или API.  
С помощью механизма схем (scheme) MPS можно запускать автоматическое выполнение задач обработки видео и отправлять уведомления о событиях в CMQ. Это позволяет отслеживать статус задач транскодирования в реальном времени. Архитектура MPS показана ниже:

![alt text](image.png)

MPS выполняет транскодирование аудио- и видеофайлов в различные битрейты и разрешения, обеспечивая плавное воспроизведение на разных устройствах при различных условиях пропускной способности сети.

Сервис поддерживает несколько типов обработки, включая:

- Транскодирование аудио и видео  
- Улучшение аудио и видео  
- Захват видео с экрана  
- Обнаружение контента  
- Модерацию контента  
- Интеллектуальный монтаж
